# Security Policy

## Reporting a Vulnerability

If you discover a security issue or a computation bug that could affect risk score accuracy, please report it.

Please do not open a public GitHub issue for security-related reports.

## Scope

This tool runs entirely client-side — no patient data is transmitted or stored. Security concerns most likely involve:

- Incorrect risk score computation
- Incorrect outcome estimates
- Issues with input validation or data handling in the browser
